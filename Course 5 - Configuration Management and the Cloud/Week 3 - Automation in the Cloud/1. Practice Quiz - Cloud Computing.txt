Practice Quiz: Cloud Computing




Q: When we use cloud services provided to the general consumer, such as Google Suite or Gmail, what cloud deployment model are we using?

A: Public cloud




Q: What is a container?

A: A virtualized environment containing applications and configurations that can run quickly and reliably on any computing environment





Q: Select the examples of Managed Web Application Platforms. (Check all that apply)

* Google App Engine.

Q: Question 4
When a company solely owns and manages its own cloud infrastructure, what type of cloud deployment model are they using?

A: Private cloud




Q: Which "direction" are we scaling when we add RAM or CPU resources to individual nodes?

A: Vertical
