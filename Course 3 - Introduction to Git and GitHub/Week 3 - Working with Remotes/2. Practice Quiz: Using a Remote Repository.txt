Practice Quiz: Using a Remote Repository


Q: In order to get the contents of a remote branch without automatically merging, which of these commands should we use?

* git remote update



Q: If we need to find more information about a remote branch, which command will help us?

* git remote show origin



Q: What command will download remote branches from remote repositories without merging the content with your current workspace automatically?

* git remote update



Q: What type of merge creates a new merge commit?

* Explicit merge



Q: What method of getting remote contents will automatically merge the remote branch with the current local branch?

* git pull
