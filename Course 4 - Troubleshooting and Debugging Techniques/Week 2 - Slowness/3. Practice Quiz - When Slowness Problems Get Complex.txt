Practice Quiz: When Slowness Problems Get Complex



Q: Which of the following can cache database queries in memory for faster processing of automated tasks?

A: Memcached




Q: What module specifies parts of a code to run in separate asynchronous events?

A: Asyncio



Q: Which of the following allows our program to run multiple instructions in parallel?

A: Threading




Q: What is the name of the field of study in computer science that concerns itself with writing programs and operations that run in parallel efficiently?

A: Concurrency




Q: What would we call a program that often leaves our CPU with little to do as it waits on data from a local disk and the Internet?

A: I/O bound
